### 项目介绍
```
知识协同管理平台是服务于各科研团队、项目团队、课题组的知识管理协同平台，
实现团队成员之间的协同开发、知识共享、知识沉淀以及创新，
提高科研和项目开发的效率。
该平台不仅能够对每位成员进行任务及进度管理，
对于团队沟通、课题开展能够提供辅助服务。
当前项目为此平台的Web用户前端。
```
